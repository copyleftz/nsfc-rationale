# NSFC Rationale Writer（理工科版）

> 国自然申请书立论依据专项写作技能 — 面向理工科（数理、化学、信息、工程、材料等）

**Author:** OpenClaw Community  
**License:** MIT

## 核心差异：理工科 vs 文管类

| 维度 | 理工科 | 文管类 |
|------|--------|--------|
| 核心逻辑 | 机理缺口 → 理论突破 | 问题导向 → 政策对齐 |
| 论证重心 | 深层物理/化学/计算机理 | 社会/行业问题解决 |

## 黄金结构：三层七段法

```
第一层：现象与意义层
    ├─ 第一段：研究背景与科学意义
    └─ 第二段：领域前沿动态与趋势

第二层：机理缺口层（★核心★）
    ├─ 第三段：国内外研究现状（技术演进视角）
    ├─ 第四段：现有方法的理论局限（机理缺口）
    └─ 第五段：本项目的理论切入点

第三层：研究框架层
    └─ 第六段：本项目研究思路与理论预期
```

## 理工科核心：机理缺口写作

| 缺口类型 | 说明 |
|---------|------|
| T1: 理论空白型 | 某物理现象缺乏理论解释 |
| T2: 模型局限型 | 模型在特定条件下精度不足 |
| T3: 尺度断裂型 | 微观与宏观理论间缺乏桥梁 |
| T4: 方法对立型 | 两种主流方法给出矛盾预测 |

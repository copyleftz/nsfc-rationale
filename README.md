# NSFC Rationale Writer

> 国自然申请书理论依据（立项依据）专项写作技能

**Author:** OpenClaw Community  
**License:** MIT

## 核心功能

- **黄金五段式模板** — 研究背景→领域重要性→国内外现状→关键缺口→本项目思路
- **Mermaid逻辑图** — 五段式结构图、缺口对应关系图、研究意义递进图
- **写作流程五步法** — 领域扫描→问题凝练→文献综述→缺口论证→整体润色
- **常见错误与修正** — 综述写成背景/缺口写得太泛/前期工作喧宾夺主等
- **外文引文比例提升攻略** — 从60%合格线到各类型项目参考标准

## 黄金五段式结构

```
第一段：研究背景（宏观面）
    ↓ 第二段：领域重要性（为什么紧迫）
    ↓ 第三段：国内外研究现状（别人做了什么）
    ↓ 第四段：关键知识缺口（别人没做什么）← 核心
    ↓ 第五段：本项目研究思路（我来做/我的独特视角）
```

## 快速开始

```bash
# 安装技能
install https://github.com/copyleftz/nsfc-rationale
```

## 内容概览

| 章节 | 字数建议 | 核心要点 |
|------|---------|---------|
| 第一段 | 200-300字 | 3句话内建立领域坐标系 |
| 第二段 | 300-400字 | 数据/政策支撑紧迫性 |
| 第三段 | 1500-2000字 | 分类归纳，国际70% |
| 第四段 | 800-1000字 | 2-3个精准缺口 |
| 第五段 | 300-400字 | 与缺口一一对应 |
